# About

- An arcade style game built with flutter have to pop as many bubbles within the time limit

# Built With
- Flutter
- Dart
- glassmorphism package
- google_fonts package

# Features
- User can click start and the game will automatically start
- Users will need to click on the bubbles on the screen within a 30 second timer
- When the timer is up the user will see the end screen with their score and a button to play again

## Getting Started

1. Clone the repository
- git clone https://github.com/desireedagondon000-gif/dreamy_arcade.git
- cd dreamy_arcade
2. Install dependencies
flutter pub get
3. Run the app
flutter run
